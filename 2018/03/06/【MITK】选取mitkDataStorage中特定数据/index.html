【MITK】选取mitkDataStorage中特定数据

发表于 2018-03-06 | 分类于 mitk

本代码用于选取mitkDataStorage中的特定数据，如特定类型、特定属性等等。

1.选取特定类型数据，如“Surface”、“Image”等类型

mitk::NodePredicateDataType::Pointer predicateDataTypeSurface = mitk::NodePredicateDataType::New("Surface");
mitk::DataStorage::SetOfObjects::ConstPointer setOfObjectsSurface = this->GetDataStorage()->GetSubset(predicateDataTypeSurface);
// transform all surface node
for (mitk::DataStorage::SetOfObjects::ConstIterator nodeIt = setOfObjectsSurface->Begin(); nodeIt != setOfObjectsSurface->End(); ++nodeIt) // for each _DataNode
{
    nodeIt.Value().GetPointer()->GetData()->GetGeometry()->SetIndexToWorldTransform(mitkTransform);
}

2.选取特定属性数据，如“name”属性为“Face”的数据

mitk::NodePredicateProperty::Pointer p = mitk::NodePredicateProperty::New("name", mitk::StringProperty::New("Face"));
mitk::DataStorage::SetOfObjects::ConstPointer rs = this->GetDataStorage()->GetSubset(p);
// Get element
if (rs->Size() >= 1)
{
    mitk::DataNode::Pointer dataNode = rs->GetElement(0);
}
