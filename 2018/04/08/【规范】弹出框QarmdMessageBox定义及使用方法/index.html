【规范】弹出框QarmdMessageBox定义及使用方法

发表于 2018-04-08 | 分类于 cpp

将QMessageBox进行封装，实现定制化的弹出框。

函数原型及说明

int QarmdMessageBox(QMessageBox::Icon icon, std::string text, std::string detailedText, QMessageBox::StandardButtons buttons = QMessageBox::Ok)

函数QarmdMessageBox包含4个参数，并返回一个int值。参数说明见图： 返回值为按钮点击值。

函数实现

int QarmdMessageBox(QMessageBox::Icon icon, std::string text, std::string detailedText, QMessageBox::StandardButtons buttons)
{
    QMessageBox message;
    message.setWindowFlags(Qt::FramelessWindowHint);
    message.setStyleSheet("QMessageBox *{font-family:黑体; color:rgb(255,255,255);}"
        "QMessageBox{background:rgb(16,69,121);border:1px solid rgb(55,159,211);border-radius:10px;}"
        "QPushButton{min-width:100px;min-height:30px;background-color:qlineargradient(spread : pad, x1 : 0, y1 : 0, x2 : 0, y2 : 1, stop : 0.457711 rgba(2, 107, 186, 255), stop : 1 rgba(0, 64, 120, 255));border-radius:10px;border:1px solid rgb(2, 91, 159);}"
        "QPushButton:hover{background-color:qlineargradient(spread : pad, x1 : 0, y1 : 0, x2 : 0, y2 : 1, stop : 0.457711 rgba(244,203,37, 255), stop : 1 rgba(179,113,8, 255));border:1px solid rgb(217, 165, 25);}");
    message.setWindowTitle(QString::fromLocal8Bit("提示"));
    message.setText(QString::fromLocal8Bit(text.c_str()));
    message.setInformativeText(QString::fromLocal8Bit(detailedText.c_str()));
    message.setIcon(icon);
    message.setStandardButtons(buttons);
    message.setButtonText(QMessageBox::Ok, QString::fromLocal8Bit("确定"));
    message.setButtonText(QMessageBox::Cancel, QString::fromLocal8Bit("取消"));
    message.setButtonText(QMessageBox::Yes, QString::fromLocal8Bit("是"));
    message.setButtonText(QMessageBox::No, QString::fromLocal8Bit("否"));
    // Pre-show message box to calculate it's actual size
    message.show();
    message.hide();
    // Set border radius
    QBitmap bmp(message.size());
    bmp.fill(Qt::white);
    QPainter p(&bmp);
    p.setPen(Qt::NoPen);
    p.setBrush(Qt::black);
    // One pixel (border width) smaller than the actual radius, to prevent cutting out border
    p.drawRoundedRect(bmp.rect(), 9, 9);
    message.setMask(bmp);
    return message.exec();
}
