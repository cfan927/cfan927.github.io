【MITK Examples】Step1-显示一个图像

发表于 2018-04-09 | 分类于 mitk

原文参考：http://docs.mitk.org/nightly/Step01Page.html

1.设置图像文件路径

Step1程序展示了如何在二维视图中显示图像，该程序必须使用图像文件Pic3D.nrrd执行。此文件所在地址为..\MITK\build\MITK-Data\Pic3D.nrrd，若文件缺失，可从http://mitk.org/download/tutorial-data/Pic3D.nrrd 下载。

在Visual Studio中设置图像文件路径：

右键单击“MitkStep1”项目，然后转到“属性->配置属性->调试”，将图像文件路径插入到“命令参数”文本字段中（..\MITK\build\MITK-Data\Pic3D.nrrd）。 设置完成后，右键单击“MitkStep1”项目，并选择“设置为启动项目”。

2.运行程序

2.1 代码结构

Part I :创建一个数据存储； Part II :从文件中读取数据； Part III:将数据放入数据存储中； Part IV:创建窗口并将数据存储传递给窗口； Part V :qt特定初始化。

2.2 程序代码

#include "QmitkRegisterClasses.h"
#include "QmitkRenderWindow.h"

#include <mitkStandaloneDataStorage.h>

#include <QApplication>
#include <itksys/SystemTools.hxx>

#include <mitkIOUtil.h>

/*加载一个nrrd格式的图像并在二维视图中显示出来*/
int main(int argc, char *argv[])    //argc是命令行总的参数个数
                                    //argv[]是argc个参数，其中第0个参数是程序的全名，
                                    //以后的参数命令行后面跟的用户输入的参数
{
  QApplication qtapplication(argc, argv);//QApplication类管理图形用户界面
                                         //应用程序的控制流和主要设置。
                                         //对于任何一个使用Qt的图形用户界面应用程序，   
                                         //都正好存在一个QApplication对象

  //打印文件名
  if (argc < 2)
  {
    fprintf(stderr, "Usage:   %s [filename] \n\n", itksys::SystemTools::GetFilenameName(argv[0]).c_str());
    return 1;
  }

  // 注册Qmitk相关的全局实例
  QmitkRegisterClasses();

  //*************************************************************************
  // Part I: 基本初始化
  //*************************************************************************

  // 创建一个数据存储
  // 数据存储可以管理所有数据对象。它被渲染机制用来渲染所有数据对象。
  // 我们使用标准的实现方法mitk::StandaloneDataStorage.
  mitk::StandaloneDataStorage::Pointer ds = mitk::StandaloneDataStorage::New();

  //*************************************************************************
  // Part II: 从文件中读取数据
  //*************************************************************************

  // 加载数据结点 (如图片文件格式、表面文件格式等)
  mitk::IOUtil::Load(argv[1], *ds);

  //*************************************************************************
  // Part IV: 创建窗口并将数据存储传递给它
  //*************************************************************************

  // 创建一个渲染窗口
  QmitkRenderWindow renderWindow;

  // 告诉渲染窗口要渲染哪一个（哪一部分）的数据存储
  renderWindow.GetRenderer()->SetDataStorage(ds);

  // 初始化渲染窗口
  mitk::TimeGeometry::Pointer geo = ds->ComputeBoundingGeometry3D(ds->GetAll());
  mitk::RenderingManager::GetInstance()->InitializeViews(geo);

  // 选择一个切面
  mitk::SliceNavigationController::Pointer sliceNaviController = renderWindow.GetSliceNavigationController();
  if (sliceNaviController)
    sliceNaviController->GetSlice()->SetPos(0);   //修改SetPos()的参数值可以得到不同
                                                  //位置的切面图像

  //*************************************************************************
  // Part V: Qt特定初始化
  //*************************************************************************
  renderWindow.show();
  renderWindow.resize(256, 256);                  //设置显示窗口大小

// 测试
#include "QtTesting.h"
  if (strcmp(argv[argc - 1], "-testing") != 0)
    return qtapplication.exec();
  else
    return QtTesting();

  // 清除：删除对DataStorage的引用。这将清除对象。
  ds = NULL;
}
