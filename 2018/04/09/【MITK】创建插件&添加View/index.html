【MITK】创建插件&添加View

发表于 2018-04-09 | 分类于 mitk

调用命令行工具：The MITK Plugin Generator，可快速创建插件。

1、创建插件方法

1) 进入MitkPluginGenerator.exe所在目录

譬如：D:\MITK 16.11\MITK-Bin\MITK-build\bin\Release

2) 打开命令行窗口

目录空白处按下shift和鼠标右键，选择“在此处打开命令窗口”

3) 输入命令

MitkPluginGenerator --plugin-symbolic-name org.mycompany.myplugin --view-name "My View"

（note: 新产生的插件默认存放在如下目录，可参考下表设置相关参数，例如：输出路径等）

C:\Users\Ariemedi02\AppData\Local\Temp\org.mycompany.myplugin

2、The MITK Plugin Generator相关参数

 [* - options are required]   // A CTK plugin generator for MITK (version 1.2.0)
-h, --help                    // Show this help text
-o, --out-dir                 // Output directory (default: /tmp)
-l, --license                 // Path to a file containing license information (default: :/MITKLicense.txt)
-v, --vendor                  // The vendor of the generated code (default: DKFZ, Medical and Biological Informatics)
-q, --quiet                   // Do not print additional information
-y, --confirm-all             // Answer all questions with 'yes'
-u, --check-update            // Check for updates and exit
-n, --no-networking           // Disable all network requests

Plugin View options
-vc, --view-class             // The View's' class name
-vn, --view-name              // * The View's human readable name

Plugin options
-ps, --plugin-symbolic-name   // * The plugin's symbolic name
-pn, --plugin-name            // The plugin's human readable name

Project options
--project-copyright           // Path to a file containing copyright information (default::/LICENSE.txt)
--project-name                // The project name
--project-app-name            // The application name

3、插件中添加new view

假设new view的名字为Useful Functions View

1、添加文件

src/internal/QmitkUsefulFunctionsView.h
src/internal/QmitkUsefulFunctionsView.cpp
Const std::string QmitkUsefulFunctionsView::VIEW_ID = "org.mitk.views.usefulfunctions";//QmitkUsefulFunctionsView.cpp文件中设置view 的id：
src/internal/QmitkUsefulFunctionsViewControls.ui
resources/UsefulFunctionsViewIcon.png
documentation/UserManual/QmitkUsefulFunctionsViewUserManual.dox
\page org_mitk_views_usefulfunctions The Useful Functions View //*.dox文件中添加。view id 作为 page id后可使用F1找到view

2、添加文件内容

files.cmake 文件添加以下内容

set(INTERNAL_CPP_FILES
  QmitkUsefulFunctionsView.cpp
)
set(UI_FILES
  src/internal/QmitkUsefulFunctionsViewControls.ui
)
set(MOC_H_FILES
  src/internal/QmitkUsefulFunctionsView.h
)
set(CACHED_RESOURCE_FILES
  resources/UsefulFunctionsViewIcon.png
)

Plugin.xml文件添加以下内容

//new view添加扩展点
<extension point="org.blueberry.ui.views">
  <view id="org.mitk.views.usefulfunctions"
        name="Useful Functions"
        class="QmitkUsefulFunctionsView"
        icon="resources/UsefulFunctionsViewIcon.png" />
</extension>
